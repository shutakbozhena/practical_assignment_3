# Practice Assignment 3

## Description

This project implements a small PostgreSQL database system for managing orders in an online store.

The database includes tables for customers, products, orders, order items, and audit logs.

The project contains:

* a function for calculating the total value of an order;
* a procedure for creating new orders;
* a procedure for adding products to an order;
* triggers that automatically update the total order amount;
* a trigger that creates audit log records after a new order is created;
* test queries demonstrating that all functions, procedures, and triggers work correctly.

the repository includes:

* `answers.md` with answers to the theory questions;
* Git commit history with meaningful commits;
* an `EXPLAIN ANALYZE` example with a short explanation(in answers.md) of the execution plan.
